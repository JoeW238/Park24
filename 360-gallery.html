📁 How to add your GoPro Max images
1.Export your 360° photos from GoPro Max as equirectangular JPEGs (use GoPro Player or GoPro Max Exporter — choose "360 Export").
2.Place all JPEG files in the same folder as this HTML file (or a subfolder, e.g. 360photos/).
3.In the scenes array in the <script> below, update each image path to match your filenames, e.g. "360photos/GSAB1234.JPG", and update the label to describe the room.
4.For thumbnails, the page automatically uses the same image file (cropped to the centre strip). No separate thumbnail needed.
5.Open the page in a browser — it must be served from a local or web server (e.g. VS Code Live Server, or upload to your hosting). Direct file:// access may block image loading due to browser security.
